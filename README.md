# 🏢 نظام المحاسبة الاحترافي v2.2.1

## 📋 نظرة عامة

نظام محاسبة شامل ومتقدم مصمم خصيصاً للشركات العربية، يدعم العملات المتعددة ويوفر أدوات محاسبية احترافية مع واجهة سهلة الاستخدام.

- 🌳 **شجرة حسابات هرمية** مع دعم مستويات متعددة

## 🛠️ التقنيات المستخدمة

- **Backend:** Laravel 10+ (PHP)
- **Frontend:** Blade Templates, Bootstrap 5, AdminLTE
- **Database:** MySQL 8.0+
- **Export:** Laravel Excel (Maatwebsite)
- **PDF:** DomPDF
- **Icons:** Font Awesome 6
- **Charts:** Chart.js

## 📦 متطلبات النظام

- PHP 8.1+
- MySQL 8.0+
- Composer
- Node.js & NPM
- Apache/Nginx

## 🚀 التثبيت

### 1. استنساخ المشروع
```bash
git clone https://github.com/altatweer/acc2.git
cd acc2
```

### 2. تثبيت التبعيات
```bash
composer install
npm install
npm run build
```

### 3. إعداد البيئة
```bash
cp .env.example .env
php artisan key:generate
```

### 4. إعداد قاعدة البيانات
```bash
# تحديث إعدادات قاعدة البيانات في .env
php artisan migrate
php artisan db:seed
```

### 5. إعداد التخزين
```bash
php artisan storage:link
```

### 6. تشغيل الخادم
```bash
php artisan serve
```

## 📚 الاستخدام

### 🏠 الواجهة الرئيسية
بعد التثبيت، يمكنك الوصول للنظام عبر:
```
http://localhost:8000
```

### 👤 بيانات الدخول الافتراضية
- **البريد الإلكتروني:** admin@example.com
- **كلمة المرور:** password

### 📊 تصدير البيانات
- **شجرة الحسابات:** `/accounts/tree`
- **التقارير المالية:** `/reports`
- **كشوف المرتبات:** `/salary-batches`

## 🎨 التخصيص

### 🖨️ إعدادات الطباعة
1. انتقل إلى **الإعدادات > إعدادات الطباعة**
2. حدد الألوان والشعارات المرغوبة
3. اختبر التصميم عبر المعاينة
4. احفظ الإعدادات

### 💰 إضافة عملات جديدة
1. انتقل إلى **الإعدادات > العملات**
2. أضف العملة الجديدة مع أسعار الصرف
3. حدد العملة الافتراضية للنظام
